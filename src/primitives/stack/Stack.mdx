# Stack Primitive

<Description of={Stories} />

The `Stack` primitive is a high-performance, flexbox layout utility extending `Box`. It enables developers to structure linear layouts (rows or columns) with calibrated spacing gaps and alignment attributes.

## Key Scenarios

### 1. Horizontal Flex Flows

Organize elements side-by-side using `direction="horizontal"`. Perfect for navigation elements, toolbars, metadata groups, or action lists.

<Canvas of={Stories.HorizontalRow} />

### 2. Spacing Ladder Gaps

Directly maps the `gap` prop to the 4px modular spacing scale (`space-1` to `space-9`), creating uniform, calibrated distance scales without manual margin paddings.
