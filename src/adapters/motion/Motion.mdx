# Motion

**Why we depend on it:** Motion (formerly Framer Motion) provides the *animation and
gesture* layer — spring physics, drag, layout animations, `AnimatePresence` for
enter/exit, and motion values that drive CSS variables. It powers the iOS feel:
swipe-to-reveal, momentum snap, and the spring presets defined in our tokens
(`--spring-snappy | smooth | bouncy | gentle`).

We currently import from the `framer-motion` package (v12).
