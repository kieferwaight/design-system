# React Aria

**Why we depend on it:** React Aria (Adobe) provides the *behavior and accessibility*
layer — keyboard interactions, focus management, ARIA semantics, pointer/press
handling, collections (GridList, ListBox), selection — without imposing any styling.
The Design Kit owns the look (Liquid Glass tokens + Motion); React Aria owns the
correctness. This split is why our components are accessible by default.

We use `react-aria-components` (the higher-level component API).
