# Spacing Foundation

Spacing is built on a source-defined modular scale.

In this design system, padding and margin are not separate token lists; instead, they are simply **spacing applied to specific box edges** (using Primitives like Box and Stack).

## Spacing Scale

The scale maps to standard `rem` units in CSS to guarantee continuous alignment with user-agent accessibility font-scaling adjustments.
